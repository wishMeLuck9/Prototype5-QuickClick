# Prototype 5 - Quick Click

Standalone Unity project for Junior Programmer Unit 5 lessons 5.1-5.4.

## Covered tutorial outcomes

- Random good and bad targets spawn from the bottom of the screen.
- Targets receive random upward force, torque, and horizontal position.
- Mouse clicks destroy targets.
- Score UI updates when good targets are clicked.
- Game over appears when a good target is missed or a bad target is clicked.
- Restart and difficulty buttons are wired through UI button listeners.

Open `Assets/Scenes/Prototype5.unity` in Unity and press Play.
